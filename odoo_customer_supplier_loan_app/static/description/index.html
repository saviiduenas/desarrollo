Customer and Supplier Loan Management

Customer and Supplier Loan Management is Designed to Assign Loans to the Customer. Customer and Supplier can Create Request for Loans which can be Approved by the Manager. Journal Entries can be Created with Installment and Loan Request.

Module Working Flow


Description

User Access Groups for Loans

Give the Required Access to the User for Loan.


Loan Proofs

With this Loan Proof Manager can Create Loan Proofs that will be Required to Approve Loan. These Loan Proofs will be Selected On Loan Types and then it will be Visible On Loan Request Form.


Loan Types

Manager can Create Differnt Loan Types for the Customers.

Option for Interest Mode : Flat Or Reducing.

Option for Repayment Method : Deduction From Payroll Or Direct Cash/Cheque.

Option for Disburse Method : Through Payroll Or Direct Cash/Cheque.


Loan Policies

Loan Policies can be Created by the Manager. Here You Get 3 Types of Policies.

1.Max Loan Amount : You can Set Maximum Amount to Apply for Loan if this Policy is Applied to Customer/Supplier Loan.

2.Gap Between Two Loans : You can Set Gap Duration in Months for Time Gap Bettwen the Loans of the Customer/Supplier.

3.Qualifying Period : You can Set Qualifying Period in Days, If this Policy is Applied to Loan then Customer/Supplier can Apply for Loan but Loan can only Get Approved After the Qualifying Period.


Loan Request

User can Create a Loan Request with Necessary Details.

Click on Apply Button to Approve Loan Request.

Approved Loan Request can be Seen by Manager for Further Process.


Compute Loan

Click on Compute Button to Compute Loan Request.


Computed Loan

Click on Disburse Button to Create Loan Disburse. Manager can Create a Loan Disburse by Giving Necessary Details of Account to the Loans.


Created Loan Disburse Journal Entry

Created Loan Disburse Journal Entry Details.


Loan Installment

Customers can Pay Multiple Installments of Loan. Create a Loan Installment and Give Amount and then Click on Pay.


Customers Loans

Details of Loan will Automatically Appear on Customers.


Customers Loan Policies

Loan Policies are also Mentioned in Customer for Better Record.


Loan Information in Website

Loan Information also Seen in Website.



Loan Request Report

Customers can Print Loan Report which will Show Full Information About the Loan.
